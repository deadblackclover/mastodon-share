# mastodon-share
Mastodon Share Button

## Installation
```bash
npm install mastodon-share
```

## Add to site
```html
<div id="mastodon-share"></div>
<script src="https://cdn.rawgit.com/deadblackclover/mastodon-share/cb49ab56/src/index.js"></script>
<script>
  getMessageMastodon("Test share button!");
</script>
```
